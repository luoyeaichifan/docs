```
在一排座位（ seats）中，1 代表有人坐在座位上，0 代表座位上是空的。

至少有一个空座位，且至少有一人坐在座位上。

亚历克斯希望坐在一个能够使他与离他最近的人之间的距离达到最大化的座位上。

返回他到离他最近的人的最大距离。

示例 1：

输入：[1,0,0,0,1,0,1]
输出：2
解释：
如果亚历克斯坐在第二个空位（seats[2]）上，他到离他最近的人的距离为 2 。
如果亚历克斯坐在其它任何一个空位上，他到离他最近的人的距离为 1 。
因此，他到离他最近的人的最大距离是 2 。 
示例 2：

输入：[1,0,0,0]
输出：3
解释： 
如果亚历克斯坐在最后一个座位上，他离最近的人有 3 个座位远。
这是可能的最大距离，所以答案是 3 。
提示：

1 <= seats.length <= 20000
seats 中只含有 0 和 1，至少有一个 0，且至少有一个 1。
```


```
这道题就是找0的个数，

1、如果在两边的话，有几个零，最大距离就是几，

2、在中间的情况有两种，偶数个零，则距离就是个数除以2，奇数个零，则距离就是个数除以2+1
```

func maxDistToClosest(seats []int) int {
	maxLeft := 0
	maxRight := 0
	maxMid := 0
	tmpMaxMid := 0
	leftIndex := 0
	maxLeftChannel :=make(chan int)
	maxRightChannel :=make(chan int)
	maxMidChannel :=make(chan int)
	rightIndex := len(seats) - 1
	go func() {
		for i := 0; i <= rightIndex; i++ {
			leftIndex = i
			if seats[i] == 0 {
				maxLeft++
			} else {
				maxLeftChannel <- maxLeft

				break
			}
		}
	}()

	go func() {
		for i := rightIndex; i >= 0; i-- {
			rightIndex = i
			if seats[i] == 0 {
				maxRight++
			} else {
				maxRightChannel <- maxRight

				break
			}
		}
	}()

	go func() {
		for i := leftIndex; i <= rightIndex; i++ {
			if seats[i] == 0 {
				tmpMaxMid++
			} else {
				if maxMid < tmpMaxMid {
					maxMid = tmpMaxMid
				}
				tmpMaxMid = 0
			}
		}
		if maxMid%2 == 0 {
			maxMid = maxMid / 2
		} else {
			maxMid = maxMid/2 + 1
		}
		maxMidChannel <- maxMid
	}()

	<-maxLeftChannel
	<-maxRightChannel
	<-maxMidChannel

	if maxLeft >= maxMid || maxRight >= maxMid {
		if maxLeft > maxRight {
			return maxLeft
		} else {
			return maxRight
		}
	} else {
		return maxMid
	}

}
